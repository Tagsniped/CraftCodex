Drop your item sprites in this folder.

The app loads sprites by filename, for example:

- enchanting_table.svg
- diamond.svg
- redstone_torch.svg

You can replace any SVG with your own SVG or PNG. If you switch to PNG, keep the same item id and update the extension in `itemArt()` inside `src/main.jsx`, or keep using SVG filenames for the easiest swap.

Recommended size: 32x32 pixels.
Recommended style: transparent background, pixel-art edges.
